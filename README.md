# employee-management-springcore
Spring Core project demonstrating IoC, Dependency Injection, XML configuration, Annotation-based configuration, @Component,@Autowired, and @Qualifier concepts.

This project demonstrates core Spring Framework concepts including:

- Inversion of Control (IoC)
- Dependency Injection (DI)
- XML-Based Configuration
- Annotation-Based Configuration
- @Component
- @Autowired
- @Qualifier
- Interface-based Programming
- Loose Coupling

The project uses an Employee/UserInfo example to explain how Spring manages objects and dependencies.
